🃏 War: The Card Game

This is actually my 1st mini project, **War: The Card Game**, a Python based game 🐍.  
The project focuses on applying Object-Oriented Programming (OOP) concepts along with a basic Graphical User Interface (GUI).

---

🛠️ Technologies Used
---> Python 🐍  
---> Tkinter (for GUI) 🎛️  
---> OOP Concepts (Classes, Objects, Methods) ⚙️  

---

🧠 Concepts Used
---> Object-Oriented Programming (Classes, Objects & Dunder Methods)  
---> GUI Design with Tkinter (Window Creation, Widgets Creation, Widget Methods Actions)  

---

✨ Features
---> Two-player gameplay  
---> Shuffled deck of cards  
---> Automatic card distribution  
---> War condition when cards are equal  
---> GUI using Tkinter  
---> Displays winner of each round
